This project is developed and tested using Jupyter Notebook in python 3.

The file "data_analysis.ipynb" contains the Exploratory Data Analysis of the movie_metadata.csv file. The python libraries pandas, numpy, matplotlib, and seaborn are used in this implementation. 

The file "ml_models.ipynb" contains the machine learning models for imdb_score prediction. The python libraries pandas, numpy, and sklearn are used in this implementation.

The results are in the "result.pdf" file.
